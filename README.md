### Postmark-PHP

**Postmark-PHP** is the _officially supported_ PHP library for [Postmark](http://postmarkapp.com).

With Postmark, you can send and _receive_ emails effortlessly.

## Requirements

- PHP 8.1, 8.2, 8.3, or 8.4
- Guzzle HTTP client
